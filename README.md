# Повтори сигнал — Rhythm Clicker

Игра на **память и реакцию** в духе Simon Says («Падающий куб»): запомни последовательность цветов и повтори её кликами. С каждым уровнем сигнал становится на один цвет длиннее, темп растёт, а ошибки отнимают жизни.

Написано на **Python 3** и **Pygame**.

## Управление

### Главное меню

| Действие | Мышь | Клавиатура |
|----------|------|------------|
| Бесконечный режим | Выбор поля **2×2** или **3×3**, затем игра | — |
| Продолжить | Кнопка «Продолжить» (если есть сохранение) | — |
| Уровни | Кнопка «Уровни» — сетка из 10 уровней (пока заглушки) | `Esc` — назад |
| Настройки | Кнопка «Настройки» | — |
| Рекорды | Кнопка «Рекорды» | — |
| Выход | Кнопка «Выход» | `Esc` (из меню) |

### Во время игры

| Действие | Мышь | Клавиатура |
|----------|------|------------|
| Повторить цвет | Клик по цветной кнопке | — |
| Пауза | Кнопка ⏸ (правый верхний угол) | `P` или `Esc` |

### Меню паузы

| Кнопка | Действие |
|--------|----------|
| Продолжить | Вернуться в игру (таймер ответа сохраняется) |
| Настройки | Открыть настройки (после «Назад» — снова пауза) |
| В главное меню | Сохранить прогресс и выйти в меню |

| Клавиша | Действие |
|---------|----------|
| `Enter` | Продолжить игру |
| `Esc` | В главное меню (с сохранением) |

### Прочее

| Экран | `Esc` |
|-------|-------|
| Настройки | Назад |
| Рекорды | В главное меню |
| Game Over | В главное меню (`Enter` тоже работает) |

Закрытие окна крестиком — выход с автосохранением текущей партии.

## Правила игры

1. Игра показывает **последовательность** из четырёх цветов: красный, синий, зелёный, жёлтый.
2. Нужно **кликнуть те же цвета в том же порядке**.
3. С каждым пройденным уровнем к сигналу добавляется **ещё один цвет**.
4. За уровень начисляются **очки** (`уровень × 10`).
5. **3 ошибки** (по умолчанию) — конец игры; количество жизней можно изменить в настройках.
6. На ответ даётся **ограниченное время** (по умолчанию 12 с).
7. После ошибки может подсветиться **правильный цвет** (если подсказки включены).
8. Каждый **5-й уровень** темп воспроизведения увеличивается (×1.5 к базовому).

## Настройки

Открываются из **главного меню** или **меню паузы**. Сохраняются в `data/settings.json`.

| Параметр | Варианты | Примечание |
|----------|----------|------------|
| Время на ответ | 8 / 12 / 16 с | Клик по строке — переключение |
| Жизни | 3 / 5 | Для **новой** игры |
| Подсказки после ошибки | Вкл / Выкл | Подсветка верного цвета |
| Разрешение окна | Выпадающий список ▼ под пунктом «Разрешение» | Применяется **сразу** |
| Полный экран | Вкл / Выкл | Переключатель отдельной строкой |

Доступные разрешения: **720×560**, **1024×768**, **1280×720**, **1600×900**, **1920×1080**, **1920×1200**.

> Изменение жизней и времени применяется к **новой** партии. Разрешение и полный экран — сразу.

## Сохранения

Все файлы данных лежат в папке `data/` (создаётся автоматически):

| Файл | Назначение |
|------|------------|
| `data/save.json` | Незавершённая игра (продолжить из меню) |
| `data/records.json` | Топ-10 рекордов с датой |
| `data/settings.json` | Пользовательские настройки |

- **Автосохранение** — каждые 5 секунд во время игры, при паузе и при выходе.
- **Рекорд** записывается при Game Over (если есть очки).

## Архитектура

Проект разделён по принципу **MVC**:

| Слой | Ответственность |
|------|-----------------|
| **Model** | Правила, фазы, очки, последовательность — без UI |
| **View** | Меню, кнопки, HUD — только отрисовка |
| **Controller** | Клавиатура/мышь, переходы между экранами, автосохранение |
| **Persistence** | Запись и чтение JSON |

View получает неизменяемый **`GameSnapshot`** и не меняет игровое состояние напрямую.

## Реализованные алгоритмы

### Базовые
- Случайная генерация последовательности
- Воспроизведение сигнала с задержкой между цветами
- Проверка ввода игрока
- Таймер на ответ

### Средние
- Ускорение темпа каждые 5 уровней (×1.5)
- Система жизней
- Подсказка после ошибки (подсветка цвета)

### Сложные
- Динамическая сложность (частые ошибки — снижение темпа; серия успехов — пропуск уровня)
- Исключение повторяющихся паттернов вида A-B-A-B
- Топ-10 рекордов с датами в файле

## Возможные проблемы

### `python` не распознаётся

Установите Python с [python.org](https://www.python.org/downloads/) и отметьте **«Add Python to PATH»**. Либо используйте:

```powershell
py main.py
```

### Ошибка `No module named 'pygame'`

Убедитесь, что зависимости установлены в том же окружении, из которого запускаете игру:

```powershell
pip install -r requirements.txt
```

### Окно не открывается / сразу закрывается

Запускайте из терминала, чтобы увидеть текст ошибки:

```powershell
cd "путь\к\проекту"
python main.py
```

### PowerShell не даёт активировать venv

```powershell
Set-ExecutionPolicy -Scope CurrentUser RemoteSigned
```

Затем снова: `.\venv\Scripts\Activate.ps1`
